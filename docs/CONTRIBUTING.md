# Руководство по внесению вклада

## Процесс работы

1. **Обсуждение** - создайте issue в Hub
2. **Планирование** - получите одобрение
3. **Разработка** - работайте в satellite репозитории
4. **Ревью** - создайте Pull Request
5. **Мерж** - после approval

## Стандарты

- Следуйте code style проекта
- Пишите тесты для нового кода
- Обновляйте документацию
